# PRD: Stratoma SDAS Landing Page

## 1. Overview
Create a professional, high-end landing page for "Stratoma SDAS" (placeholder name based on user request). Since no specific content was provided, this will be a generic but polished "Digital Transformation & AI" corporate profile, aligned with Stratoma's brand (High-End Minimalist, Professional Corporate).

## 2. Design Aesthetics
- **Style:** Professional Corporate, High-End Minimalist.
- **Colors:** Deep navy blues, charcoal greys, clean white, subtle metallic accents (silver/gold/bronze).
- **Typography:** Clean sans-serif (Inter, Roboto, or similar) for readability and modern feel.
- **Layout:**
    - **Hero Section:** Impactful headline, subheadline, "Contact Us" CTA. Background: Subtle abstract geometric pattern or high-quality office/tech placeholder.
    - **Features/Services:** 3-4 column grid with icons (SVG) describing generic high-tech services (AI Integration, Cloud Solutions, Data Analytics).
    - **About Us:** Brief section establishing authority.
    - **Contact:** Functional WhatsApp button (as per mandatory rules) and a simple contact form email link.
    - **Footer:** Copyright, links.

## 3. Functional Requirements
- **Responsive:** Mobile-first design.
- **WhatsApp Integration:** Floating or fixed button for direct contact.
- **Performance:** Lightweight, fast loading (no heavy frameworks, pure HTML/Tailwind or minimal CSS).
- **SEO:** Basic meta tags for title and description.

## 4. Technical Stack
- **Framework:** Pure HTML + Tailwind CSS (via CDN for simplicity in single file) or Stitch-generated assets.
- **Deployment:** Docker (Nginx/Alpine).
- **Hosting:** Coolify (Hetzner VPS).
